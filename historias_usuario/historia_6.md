# Historia de Usuario 6: Historial de Pedidos

## Descripción

Como cliente frecuente de LePetitDep, quiero consultar mis pedidos anteriores, para llevar un seguimiento de mis compras y revisar órdenes realizadas anteriormente.

## Criterios de aceptación

- El usuario debe poder visualizar todos sus pedidos anteriores.
- Cada pedido debe mostrar la fecha, los productos solicitados y el total.
- El número de orden como identificación de factura debe estar disponible para consulta.
- Solo usuarios autenticados pueden acceder al historial.
- La información debe mostrarse de manera clara y organizada.
