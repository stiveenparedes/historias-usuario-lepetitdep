# Historia de Usuario 5: Realizar Pedido

## Descripción

Como cliente, quiero confirmar mi pedido y seleccionar una forma de pago,
para recibir un número de orden y recoger mis productos en el local.

## Criterios de aceptación

- El sistema debe mostrar un resumen completo del pedido.
- El usuario debe poder confirmar la compra.
- El sistema debe permitir seleccionar un método de pago simulado.
- Una vez confirmado, debe generarse un número de orden único.
- El sistema debe mostrar un mensaje indicando que el pedido fue realizado exitosamente.
