# Historia de Usuario 2: Inicio de Sesión

## Descripción

Como cliente registrado, quiero iniciar sesión con mis credenciales, para acceder a mi cuenta y poder continuar realizando pedidos sin necesidad de registrarme nuevamente.

## Criterios de aceptación

- El usuario debe ingresar su correo electrónico y contraseña.
- El sistema debe validar la información proporcionada.
- Si las credenciales son correctas, el acceso debe ser concedido.
- Si existe algún error, el sistema debe informar al usuario.
- Solo los usuarios registrados pueden acceder a su cuenta.
