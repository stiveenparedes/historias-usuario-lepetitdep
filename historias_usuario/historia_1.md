# Historia de Usuario 1: Registro de Usuario

## Descripción

Como persona interesada en comprar café y productos de panadería en LePetitDep, quiero crear una cuenta utilizando mi correo electrónico y una contraseña, para poder acceder a la plataforma y realizar pedidos de forma rápida y segura.

## Criterios de aceptación

- El usuario debe poder registrarse con un correo electrónico que sea válido.
- El usuario debe crear una contraseña para poder proteger su cuenta.
- El sistema debe verificar que el correo no esté registrado en el sistema.
- Al finalizar el registro, el sistema debe mostrar un mensaje de bienvenida al usuario.
- El usuario debe poder iniciar sesión con la cuenta creada con sus credenciales.
