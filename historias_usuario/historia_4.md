# Historia de Usuario 4: Agregar Productos al Carrito

## Descripción

Como cliente, quiero agregar al carrito los productos que quiero comprar y definir la cantidad de cada uno, para de esta manera organizar mi pedido antes de confirmarlo.

## Criterios de aceptación

- El usuario debe poder seleccionar uno o varios productos.
- El usuario debe poder indicar la cantidad deseada.
- El carrito debe mostrar los productos agregados.
- El total del pedido debe actualizarse automáticamente.
- El usuario debe poder modificar o eliminar productos del carrito.
