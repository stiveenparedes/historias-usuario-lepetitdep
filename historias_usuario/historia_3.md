# Historia de Usuario 3: Explorar Menú

## Descripción

Como cliente de LePetitDep, quiero explorar el menú de cafés y productos de panadería, para conocer las opciones disponibles y elegir lo que deseo comprar.

## Criterios de aceptación

- El sistema debe mostrar todos los productos disponibles.
- Cada producto debe incluir nombre, descripción y precio.
- El usuario debe poder navegar fácilmente por el menú.
- La información de los productos debe mostrarse de forma clara y detallad.
- El menú debe estar disponible para consulta antes de realizar un pedido.
