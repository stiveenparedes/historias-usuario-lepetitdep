# historias-usuario-lepetitdep

## Descripción

LePetitDep es una aplicación web ficticia que permite a los usuarios pedir café y productos de panadería para recoger en su local.

## Funcionalidades

- Registro de usuario
- Inicio de sesión
- Explorar menú
- Agregar productos al carrito
- Realizar pedido
- Historial de pedidos
